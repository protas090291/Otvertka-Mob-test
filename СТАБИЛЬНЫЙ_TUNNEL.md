# 🚀 Стабильный Tunnel - Решение всех проблем

## ✅ Что делает этот скрипт:

1. **Автоматически перезапускает tunnel** при обрыве
2. **Сохраняет tunnel URL** при первом получении
3. **Генерирует QR код** автоматически
4. **Работает через любую сеть** без пересканирования

---

## 🎯 Быстрый старт:

### Запуск стабильного tunnel:

```bash
cd Helper2/mobile-app
./start-tunnel-stable.sh
```

Или через npm:
```bash
npm run start:tunnel:stable
```

### Получение QR кода:

```bash
./get-qr-stable.sh
```

Или через npm:
```bash
npm run qr:stable
```

---

## 📋 Как это работает:

1. **Запуск:** Скрипт запускает tunnel с автоматическим перезапуском
2. **Сохранение URL:** При получении tunnel URL сохраняет его в файлы:
   - `tunnel-url.txt` - текущий URL
   - `tunnel-url-saved.txt` - история всех URL
3. **QR код:** Автоматически генерирует и открывает QR код
4. **Автоперезапуск:** При обрыве tunnel автоматически перезапускается

---

## 💡 Преимущества:

- ✅ **Не истекает** - автоматический перезапуск при обрыве
- ✅ **Работает с любой сети** - использует tunnel URL (`exp.direct`)
- ✅ **Не нужно пересканировать** - один QR код работает всегда
- ✅ **Сохраняет URL** - можно получить QR код в любой момент
- ✅ **Стабильный** - работает в фоне через screen

---

## 📱 Использование:

### Шаг 1: Запустите tunnel
```bash
./start-tunnel-stable.sh
```

Подождите 2-3 минуты пока появится QR код.

### Шаг 2: Отсканируйте QR код
QR код откроется автоматически в браузере. Отсканируйте его в Expo Go.

### Шаг 3: Готово!
Теперь можно:
- Переключаться между сетями (Wi-Fi ↔ мобильный интернет)
- Не нужно пересканировать QR код
- Tunnel автоматически перезапустится при обрыве

---

## 🔄 Если нужно получить QR код снова:

```bash
./get-qr-stable.sh
```

Или откройте файл `qr-code-url.txt` - там будет ссылка на QR код.

---

## 🛑 Остановка:

```bash
./stop-expo.sh
```

---

## 📋 Файлы:

- `tunnel-url.txt` - текущий tunnel URL
- `tunnel-url-saved.txt` - история всех tunnel URL
- `qr-code-url.txt` - ссылка на QR код
- `expo.log` - логи tunnel

---

## ⚠️ Важно:

- Tunnel URL может измениться при полном перезапуске (если ngrok выдаст новый)
- Но пока tunnel работает, URL остается стабильным
- При автоматическом перезапуске (после обрыва) URL обычно сохраняется
- Если URL изменился, используйте `./get-qr-stable.sh` для получения нового QR кода
