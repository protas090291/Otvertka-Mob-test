# Отвёртка - Мобильное приложение

Мобильное приложение для управления строительством на базе React Native и Expo.

## 🚀 Возможности

- 📱 Управление дефектами
- 🏢 Управление проектами и квартирами
- 🎤 Голосовое управление
- 📊 Статистика и отчеты
- 👥 Управление пользователями и ролями
- 📸 Загрузка фотографий дефектов
- 🔐 Авторизация через Supabase

## 📋 Требования

- Node.js 18+
- npm или yarn
- Expo CLI
- iOS Simulator (для iOS) или Android Emulator (для Android)

## 🛠️ Установка

```bash
# Установка зависимостей
npm install

# Запуск приложения
npm start
```

## 📱 Запуск на устройстве

### iOS
```bash
npm run ios
```

### Android
```bash
npm run android
```

## 🌐 Запуск с туннелем (для тестирования с разных сетей)

```bash
# Запуск с Cloudflare Tunnel (бесплатный, стабильный)
npm run start:cloudflare

# Запуск с ngrok tunnel
npm run start:tunnel
```

## 🔧 Конфигурация

### Supabase
Настройки Supabase находятся в `lib/supabase.ts`. Убедитесь, что у вас есть правильные ключи доступа.

### Expo
Конфигурация Expo находится в `app.json`. Здесь можно настроить:
- Название приложения
- Иконки и splash screen
- Разрешения (микрофон для голосового управления)
- Версию приложения

## 🎤 Голосовое управление

Приложение поддерживает голосовое управление для:
- Навигации к дефектам конкретной квартиры: "Перейди в дефекты квартиры 203"
- Создания дефекта: "Создай дефект в квартире 404"

**Примечание:** Голосовое управление работает только в standalone builds, не в Expo Go.

## 🐛 Решение проблем

### Проблемы с Android
Если возникают проблемы с "zero" ошибкой на Android, убедитесь, что:
- Polyfills загружены первыми (проверьте `App.tsx`)
- Все зависимости установлены (`npm install`)

### Проблемы с сетью
Если приложение не загружается с мобильного интернета:
- Используйте tunnel режим: `npm run start:cloudflare`
- Убедитесь, что Metro Bundler запущен и доступен
