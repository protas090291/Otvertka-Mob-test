# 🔧 Решение проблемы с разными сетями

## ⚠️ Проблема
Приложение работает через общий Wi-Fi, но при переключении на мобильный интернет выдает ошибку подключения.

## ✅ Решение

### Шаг 1: Убедитесь что используете tunnel URL

Tunnel URL должен содержать `exp.direct`:
```
exp://xxxx-xxxx.exp.direct:80
```

**НЕ локальный адрес:**
```
exp://192.168.x.x:8081  ❌
```

### Шаг 2: Получите актуальный tunnel URL

```bash
cd Helper2/mobile-app
./get-tunnel-url.sh
```

Или подключитесь к screen:
```bash
screen -r expo-tunnel
```

### Шаг 3: Пересканируйте QR код

**Важно:** После переключения сети нужно **пересканировать QR код**!

1. Откройте Expo Go на телефоне
2. Отсканируйте **новый QR код** с tunnel URL
3. Приложение подключится через интернет

### Шаг 4: Проверьте что tunnel работает

```bash
cd Helper2/mobile-app
ps aux | grep ngrok
```

Должен быть процесс ngrok.

---

## 🔄 Если все равно не работает:

### Вариант 1: Перезапустите tunnel

```bash
cd Helper2/mobile-app
./stop-expo.sh
./start-tunnel.sh
```

Подождите 2-3 минуты и пересканируйте QR код.

### Вариант 2: Проверьте что tunnel URL правильный

```bash
cd Helper2/mobile-app
./get-tunnel-url.sh
```

URL должен содержать `exp.direct`.

### Вариант 3: Запустите напрямую в терминале

```bash
cd Helper2/mobile-app
./stop-expo.sh
npx expo start --tunnel --clear
```

Дождитесь QR кода и пересканируйте его.

---

## 💡 Важно понимать:

1. **Tunnel URL** (`exp.direct`) работает через интернет - можно переключаться между сетями
2. **После переключения сети** нужно пересканировать QR код
3. **Локальный URL** (`192.168.x.x`) работает только в одной Wi-Fi сети

---

## 📱 Правильная последовательность:

1. ✅ Запустите tunnel: `./start-tunnel.sh`
2. ✅ Получите QR код с tunnel URL (`exp.direct`)
3. ✅ Отсканируйте QR код в Expo Go
4. ✅ Подключитесь через Wi-Fi - работает ✅
5. ✅ Переключитесь на мобильный интернет
6. ✅ **Пересканируйте QR код** - теперь работает через интернет ✅

---

**Главное: используйте tunnel URL (`exp.direct`) и пересканируйте QR код после переключения сети!**
