# 🔧 Обновление переменных окружения в Time Web Cloud

## ✅ Что было изменено

Metro Bundler теперь использует порт 8080 (PORT) вместо 8080 (METRO_PORT).

## 📋 Что нужно сделать в Time Web Cloud

### 1. Обновите переменные окружения

В разделе "Настройки" → "Переменные":

1. **Удалите переменную `METRO_PORT`** (она больше не нужна)
2. **Оставьте переменную `PORT`** со значением `8080`
3. **Остальные переменные оставьте как есть:**
   - `NODE_ENV` = `production`
   - `EXPO_NO_DOTENV` = `1`
   - `EXPO_DEVTOOLS_LISTEN_ADDRESS` = `0.0.0.0`
   - `NODE_OPTIONS` = `--max-old-space-size=4096`

### 2. Сохраните и перезапустите

1. Нажмите "Сохранить данные"
2. Перезапустите приложение
3. Подождите 2-3 минуты

### 3. Проверьте QR-код

После перезапуска:
1. Откройте QR-код: `https://protas090291-otvertka-mob-test-beaf.twc1.net/qr`
2. QR-код теперь использует порт 8080
3. Отсканируйте QR-код в Expo Go
4. Приложение должно подключиться!

## ✅ Результат

- Metro Bundler работает на порту 8080 (открыт извне)
- Express сервер работает на порту 8080 (для health check)
- Expo Go может подключиться к порту 8080
- Не нужно открывать дополнительный порт 8081
