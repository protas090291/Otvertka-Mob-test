# 🔧 Как открыть порт 8081 в Time Web Cloud

## ⚠️ Важно

Expo Go требует прямой доступ к Metro Bundler на порту 8081. Туннели не используются.

## 📋 Где найти настройки портов в Time Web Cloud

### Вариант 1: В разделе "Сеть"

1. Откройте ваше приложение `otvertka-mob`
2. Перейдите в раздел **"Сеть"** (Network)
3. Найдите раздел **"Порты"** (Ports) или **"Firewall"**
4. Добавьте порт 8081

### Вариант 2: В разделе "Настройки"

1. Откройте ваше приложение
2. Перейдите в **"Настройки"** (Settings)
3. Найдите раздел **"Сеть"** или **"Порты"**
4. Добавьте порт 8081

### Вариант 3: В разделе "Деплой"

1. Откройте ваше приложение
2. Перейдите в **"Деплой"** (Deploy)
3. Найдите настройки сети или портов
4. Добавьте порт 8081

## 🔍 Что искать

Ищите:
- **"Порты"** (Ports)
- **"Firewall"** (Брандмауэр)
- **"Открытые порты"** (Open ports)
- **"Публичные порты"** (Public ports)

## ✅ Настройки порта 8081

Когда найдете раздел с портами, добавьте:
- **Порт:** `8081`
- **Протокол:** `TCP`
- **Тип:** `Публичный` или `Public`
- **Описание:** `Metro Bundler для Expo Go`

## 🔄 После добавления порта

1. Сохраните изменения
2. Перезапустите приложение
3. Подождите 1-2 минуты
4. Проверьте QR-код: `https://protas090291-otvertka-mob-test-beaf.twc1.net/qr`
5. Отсканируйте QR-код в Expo Go

## ❓ Если не можете найти настройки портов

В Time Web Cloud порты могут настраиваться автоматически на основе команды запуска. Проверьте:

1. В разделе "Настройки" → "Команда запуска" должно быть: `node server.js`
2. В разделе "Настройки" → "Переменные окружения" должны быть:
   - `PORT` = `8080`
   - `METRO_PORT` = `8081`

Если порт 8081 не открывается автоматически, возможно нужно обратиться в поддержку Time Web Cloud.

---

**После открытия порта 8081 Expo Go должен подключиться!**
