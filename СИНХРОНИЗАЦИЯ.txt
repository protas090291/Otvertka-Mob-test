🎯 КАК ЭТО РАБОТАЕТ:

1. Вы редактируете код в редакторе (VS Code, Cursor и т.д.)
   📁 Например: screens/DashboardScreen.tsx

2. Сохраняете файл (Cmd+S или Ctrl+S)
   💾 Файл сохраняется на диск

3. Metro Bundler (Expo сервер) автоматически:
   🔍 Обнаруживает изменение файла
   ⚡ Перекомпилирует измененный код
   📤 Отправляет обновление в Expo Go

4. Expo Go на телефоне:
   📱 Получает обновление
   🔄 Автоматически обновляет экран
   ✨ Показывает новые изменения!

ПРИМЕР 1: Изменение текста
───────────────────────────
1. Откройте: screens/DashboardScreen.tsx
2. Найдите: "Добро пожаловать в Отвёртку"
3. Измените на: "Привет! Это мое приложение"
4. Сохраните (Cmd+S)
5. Через 1-2 сек на телефоне увидите новый текст!

ПРИМЕР 2: Изменение цвета
───────────────────────────
1. Откройте: constants/Colors.ts
2. Измените: primary: '#3b82f6' на primary: '#ff0000'
3. Сохраните
4. Через 1-2 сек все синие элементы станут красными!

ПРИМЕР 3: Добавление элемента
──────────────────────────────
1. Откройте: screens/DashboardScreen.tsx
2. Добавьте новый <Text> компонент
3. Сохраните
4. Через 1-2 сек новый элемент появится на экране!
