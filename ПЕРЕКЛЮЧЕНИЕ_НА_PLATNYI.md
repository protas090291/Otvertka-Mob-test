# 🔄 Переключение на платный вариант (Ngrok Pro)

## ⚠️ Если Cloudflare Tunnel вызывает проблемы:

Если бесплатный Cloudflare Tunnel не работает стабильно или возникают проблемы, переключитесь на платный Ngrok Pro.

---

## 🚀 Быстрое переключение на Ngrok Pro:

### Шаг 1: Остановите Cloudflare Tunnel

```bash
cd Helper2/mobile-app
./stop-expo.sh
```

### Шаг 2: Зарегистрируйтесь на Ngrok Pro

1. Откройте https://dashboard.ngrok.com
2. Зарегистрируйтесь или войдите
3. Выберите план **Pro** ($8/месяц)
4. Перейдите в раздел **"Your Authtoken"**
5. Скопируйте ваш authtoken

### Шаг 3: Установите authtoken

```bash
ngrok config add-authtoken YOUR_AUTHTOKEN_HERE
```

### Шаг 4: Запустите Ngrok Pro

```bash
cd Helper2/mobile-app
npm run start:ngrok-pro
```

Или напрямую:

```bash
./start-ngrok-pro-stable.sh
```

### Шаг 5: Отсканируйте QR код

QR код откроется автоматически в браузере. Отсканируйте его в Expo Go.

## 🔧 Если Ngrok Pro тоже не работает:

1. Проверьте интернет соединение
2. Проверьте firewall настройки
3. Убедитесь что authtoken правильный
4. Проверьте логи: `tail -f ngrok.log`
