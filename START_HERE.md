## 🎯 Как запустить:

### Шаг 1: Перейдите в папку приложения
```bash
cd Helper2/mobile-app
```

### Шаг 2: Запустите приложение
```bash
npm start
```

### Шаг 3: Выберите способ запуска:

**На телефоне (рекомендуется):**
1. Установите **Expo Go** из App Store или Google Play
2. Отсканируйте QR-код, который появится в терминале

**На симуляторе:**
- iOS: Нажмите `i` в терминале (требуется macOS и Xcode)
- Android: Нажмите `a` в терминале (требуется Android Studio)

**В браузере:**
- Нажмите `w` в терминале

## ⚠️ Важно:

Если видите ошибки при запуске:
1. Убедитесь, что Node.js версии 18+ установлен
2. Очистите кеш: `npx expo start --clear`
3. Переустановите зависимости: `rm -rf node_modules && npm install`
