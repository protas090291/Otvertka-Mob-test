# 🔧 Решение проблемы переключения сети

## ⚠️ Проблема:

Приложение работает на Wi-Fi, но при переключении на мобильный интернет перестает загружаться.

## 🔍 Причина:

**Expo Go кэширует локальный адрес** когда телефон и ноутбук в одной сети. При переключении на другую сеть Expo Go пытается использовать старый локальный адрес вместо tunnel URL.

---

## ✅ Решение 1: Пересканировать QR код (бесплатно)

### Шаги:

1. **Отсканируйте QR код на Wi-Fi** и дождитесь загрузки (100%)
2. **Переключите телефон на мобильный интернет**
3. **ОБЯЗАТЕЛЬНО пересканируйте QR код заново** - используйте тот же tunnel URL
4. **Дождитесь загрузки** - теперь будет работать через мобильный интернет

### Почему это работает:

- При первом сканировании Expo Go запоминает локальный адрес
- При переключении сети локальный адрес недоступен
- Пересканирование заставляет Expo Go использовать tunnel URL

---

## 🔍 Проверка:

### Убедитесь что используете tunnel URL:

QR код должен содержать:
```
exp://xxxx-xxxx.exp.direct
```

**НЕ используйте:**
```
exp://192.168.x.x:8081  ❌ (локальный адрес)
exp://localhost:8081    ❌ (локальный адрес)
```
