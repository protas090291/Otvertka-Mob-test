# 🔥 Hot Reload и Fast Refresh в Expo

## ✅ Автоматическое обновление работает!

Expo Go поддерживает **Fast Refresh** (Hot Reload) по умолчанию. Это значит:

### 🎯 Что происходит автоматически:

1. **Изменения в коде** → Автоматически обновляются в приложении
2. **Сохранение файла** → Приложение обновляется за 1-2 секунды
3. **Не нужно перезапускать** → Все работает автоматически!

### 📝 Как это работает:

1. Откройте файл в редакторе (например, `screens/DashboardScreen.tsx`)
2. Внесите изменения (измените текст, цвета, стили)
3. **Сохраните файл** (Cmd+S или Ctrl+S)
4. Через 1-2 секунды изменения появятся в Expo Go на телефоне!

### ⚡ Что обновляется автоматически:

✅ Изменения в компонентах React
✅ Изменения стилей (StyleSheet)
✅ Изменения текста
✅ Изменения цветов и размеров
✅ Добавление/удаление элементов
✅ Изменения в навигации (частично)

### 🔄 Когда нужна полная перезагрузка:

❌ Изменения в `package.json` (нужно перезапустить Expo)
❌ Изменения в `app.json` (нужно перезапустить Expo)
❌ Изменения в `babel.config.js` (нужно перезапустить Expo)
❌ Установка новых пакетов (нужно перезапустить Expo)
❌ Изменения в нативных модулях

### 🛠️ Команды для перезапуска:

```bash
# Остановить Expo (Ctrl+C в терминале)
# Затем запустить заново:
npx expo start --lan --clear
```

### 💡 Полезные советы:

1. **Держите терминал открытым** - там видны ошибки и предупреждения
2. **Смотрите на телефон** - изменения появляются сразу
3. **Если не обновляется** - нажмите "Reload JS" в Expo Go
4. **Ошибки в коде** - показываются красным экраном с описанием

### 🎨 Пример работы:

1. Откройте `screens/DashboardScreen.tsx`
2. Измените текст "Добро пожаловать" на "Привет!"
3. Сохраните файл (Cmd+S)
4. Через секунду на телефоне увидите "Привет!" вместо "Добро пожаловать"
