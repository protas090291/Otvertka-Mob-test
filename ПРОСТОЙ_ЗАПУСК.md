# 📱 Простой способ получить QR код

## Проблема
Автоматический фоновый запуск не всегда показывает QR код сразу.

## ✅ Решение: Запуск в отдельном терминале

### Шаг 1: Откройте новый терминал

### Шаг 2: Перейдите в директорию проекта
```bash
cd Helper2/mobile-app
```

### Шаг 3: Запустите Expo
```bash
npx expo start --tunnel --clear
```

### Шаг 4: Дождитесь QR кода
QR код появится в терминале через 2-3 минуты после запуска Metro Bundler.

### Шаг 5: Отсканируйте QR код
Откройте Expo Go на телефоне и отсканируйте QR код из терминала.

---

## 💡 Альтернатива: Сохранить QR код в файл

Если хотите сохранить QR код в файл:

```bash
cd Helper2/mobile-app
npx expo start --tunnel --clear 2>&1 | tee expo.log
```

QR код будет в файле `expo.log`.

---

## ⚠️ Важно

- **Не закрывайте терминал** пока используете приложение
- QR код появится через 2-3 минуты после запуска
- Если tunnel оборвется, перезапустите команду

---

## 🚀 Для фонового запуска (после получения QR кода)

После того как получили QR код и убедились что все работает, можете использовать фоновый запуск:

```bash
cd Helper2/mobile-app
./start-background.sh
```

Но для первого запуска и получения QR кода лучше использовать прямой запуск в терминале.
