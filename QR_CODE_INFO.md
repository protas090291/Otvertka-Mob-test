# 📱 Информация о QR-коде

## ✅ Приложение запущено!

Expo сервер работает на: **http://localhost:8081**

## 🔍 Как получить QR-код:

### Способ 1: Откройте в браузере
Откройте в браузере: **http://localhost:8081**

Там вы увидите веб-интерфейс Expo с QR-кодом.

### Способ 2: Проверьте терминал
QR-код должен появиться в терминале после полного запуска Metro Bundler.

### Способ 3: Используйте URL напрямую
В приложении Expo Go на телефоне:
1. Откройте Expo Go
2. Нажмите "Enter URL manually"
3. Введите: `exp://localhost:8081`

Или если вы в той же сети Wi-Fi, используйте IP вашего компьютера:
- Найдите IP: `ifconfig | grep "inet " | grep -v 127.0.0.1`
- Используйте: `exp://ВАШ_IP:8081`

## 📱 Быстрый способ подключения:

1. Убедитесь, что телефон и компьютер в одной Wi-Fi сети
2. Откройте Expo Go на телефоне
3. Нажмите "Scan QR code"
4. Откройте в браузере: http://localhost:8081
5. Отсканируйте QR-код с веб-страницы

## 🔧 Если QR-код не появляется:

Проверьте, что процесс запущен:
```bash
ps aux | grep "expo start"
```

Если процесс не запущен, запустите:
```bash
cd Helper2/mobile-app
npx expo start --clear
```
