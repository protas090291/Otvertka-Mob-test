## ⚠️ Если tunnel обрывается:

### Вариант 1: Перезапустить текущий tunnel

```bash
cd Helper2/mobile-app
./stop-expo.sh
npm run start:tunnel:stable
npm run qr:stable
```

### Вариант 2: Переключиться на Ngrok Pro (платный, самый стабильный)

```bash
cd Helper2/mobile-app

# 1. Зарегистрируйтесь на https://dashboard.ngrok.com
# 2. Выберите план Pro ($8/месяц)
# 3. Получите authtoken
# 4. Установите:
ngrok config add-authtoken YOUR_TOKEN

# 5. Запустите:
npm run start:ngrok-pro
```

**Ngrok Pro даст:**
- ✅ 99.9% стабильность
- ✅ Без обрывов
- ✅ Высокая скорость
- ✅ Работает 24/7
