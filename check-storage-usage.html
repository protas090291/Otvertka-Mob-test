📊 Лимиты Supabase (справочная информация)

Бесплатный план (Free):
• Хранилище: 1 GB
• Трафик: 2 GB/месяц
• Размер файла: до 50 MB

Pro план ($25/месяц):
• Хранилище: 100 GB
• Трафик: 250 GB/месяц
• Размер файла: до 50 MB

Как проверить ваш план:
1. Откройте Supabase Dashboard
2. Settings → Billing
3. Посмотрите ваш текущий план

🔧 Как проверить использование в Dashboard

Шаг 1: Проверка использования хранилища
1. Откройте Supabase Dashboard
2. Выберите ваш проект
3. Перейдите в Settings → Usage (или Usage в боковом меню)
4. Посмотрите раздел Storage
5. Там будет показано использованное хранилище и лимит

Шаг 2: Проверка трафика
1. В том же разделе Usage
2. Посмотрите раздел Bandwidth
3. Там будет показан использованный трафик и лимит

Шаг 3: Если лимит превышен
• Обновите план Supabase
• Или удалите старые файлы из Storage
• Или дождитесь обновления лимита (месячный цикл)
