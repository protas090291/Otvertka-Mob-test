📝 Инструкции для Supabase Dashboard

Как проверить вручную:

1. Проверка bucket:
- Откройте Supabase Dashboard
- Перейдите в Storage → Buckets
- Найдите bucket defect-photos
- Проверьте, что он существует и настроен правильно

2. Проверка публичного доступа:
- В настройках bucket проверьте "Public bucket"
- Если bucket приватный, нужны правильные RLS политики

3. Проверка RLS политик:
- Storage → Policies
- Должны быть политики для:
• SELECT (чтение файлов)
• INSERT (загрузка файлов)
• UPDATE (обновление файлов)
• DELETE (удаление файлов)

4. Проверка ограничений:
- Settings → API → Storage
- Проверьте максимальный размер файла
- Проверьте лимиты трафика (если есть)

5. Проверка существующих файлов:
- Storage → defect-photos
- Посмотрите список загруженных файлов
- Проверьте их размер и дату загрузки
