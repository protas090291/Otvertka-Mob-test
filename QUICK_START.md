# Быстрый старт

## ✅ Установка завершена!

Зависимости установлены, приложение готово к запуску.

## 🚀 Запуск приложения

### Вариант 1: Через Expo Go (рекомендуется для тестирования)

1. Установите приложение **Expo Go** на ваш телефон:
   - [iOS App Store](https://apps.apple.com/app/expo-go/id982107779)
   - [Google Play Store](https://play.google.com/store/apps/details?id=host.exp.exponent)

2. Запустите сервер разработки:
   ```bash
   npm start
   ```

3. Отсканируйте QR-код, который появится в терминале или браузере

### Вариант 2: iOS симулятор (только на macOS)

```bash
npm run ios
```

### Вариант 3: Android эмулятор

```bash
npm run android
```

### Вариант 4: Веб-браузер

```bash
npm run web
```

## 📱 Что вы увидите

После запуска приложение откроется с:
- **Главный экран (Dashboard)** - статистика, быстрые действия, проекты
- **Проекты** - список активных проектов
- **График** - задачи и расписание (в разработке)
- **Бюджет** - управление бюджетом (в разработке)
- **Материалы** - склад и материалы (в разработке)
- **Дефекты** - отслеживание дефектов (в разработке)
- **Отчёты** - отчёты и документы (в разработке)
- **Профиль** - настройки пользователя

## 🔧 Устранение проблем

### Ошибка "Unable to resolve module"
```bash
npm install
```

### Ошибка с Metro bundler
```bash
npx expo start --clear
```

### Проблемы с кешем
```bash
rm -rf node_modules
npm install
npx expo start --clear
```
