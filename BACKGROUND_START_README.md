# 🚀 Фоновый запуск Expo с Tunnel режимом

Этот набор скриптов позволяет запускать Expo в фоновом режиме с tunnel, чтобы:
- ✅ Приложение продолжало работать после закрытия терминала
- ✅ Другие люди могли тестировать приложение через интернет
- ✅ Не требовался Expo аккаунт у тестировщиков

## 📋 Требования

1. **Expo аккаунт** (один на сервере) - уже создан ✅
2. **Вход в аккаунт**: `npx expo login` - уже выполнен ✅

## 🎯 Быстрый старт

### Запуск в фоновом режиме:
```bash
./start-background.sh
# или
npm run start:background
```

### Просмотр QR кода:
```bash
./show-qr.sh
# или
npm run qr
```

### Просмотр логов:
```bash
./view-logs.sh
# или
npm run logs
```

### Остановка:
```bash
./stop-expo.sh
# или
npm run stop
```

## 📱 Для тестировщиков

1. Установите **Expo Go** на телефон:
   - [App Store (iOS)](https://apps.apple.com/app/expo-go/id982107779)
   - [Google Play (Android)](https://play.google.com/store/apps/details?id=host.exp.exponent)

2. Откройте Expo Go и отсканируйте QR код (или откройте ссылку)

3. **Не требуется Expo аккаунт** - просто сканируйте QR код!

## 🔧 Доступные команды

| Команда | Описание |
|---------|----------|
| `./start-background.sh` | Запустить Expo в фоне с tunnel |
| `./stop-expo.sh` | Остановить Expo |
| `./view-logs.sh` | Смотреть логи в реальном времени |
| `./show-qr.sh` | Показать QR код для подключения |
| `npm run start:background` | То же что `./start-background.sh` |
| `npm run stop` | То же что `./stop-expo.sh` |
| `npm run logs` | То же что `./view-logs.sh` |
| `npm run qr` | То же что `./show-qr.sh` |

## 📝 Файлы

- `expo.log` - логи работы Expo (здесь будет QR код)
- `expo.pid` - PID процесса (для остановки)

## ⚠️ Важно

1. **Первый запуск tunnel** может занять 30-60 секунд
2. **QR код** появится в логах после запуска
3. **Процесс продолжит работать** после закрытия терминала
4. **Тестировщикам нужен только Expo Go** (без аккаунта)

## 🐛 Решение проблем

### Если tunnel не запускается:
```bash
# Проверьте, что вы вошли в аккаунт
npx expo whoami

# Если не вошли, войдите
npx expo login
```

### Если процесс не останавливается:
```bash
# Найти процесс вручную
ps aux | grep "expo start"

# Убить процесс
pkill -f "expo start"
```

### Если QR код не виден:
```bash
# Откройте файл логов напрямую
cat expo.log | grep -A 10 "Metro\|exp://"
```

## 💡 Альтернативные режимы

### LAN режим (только локальная сеть, без аккаунта):
```bash
npx expo start --lan --clear
```

### Обычный режим (только локально):
```bash
npx expo start --clear
```
